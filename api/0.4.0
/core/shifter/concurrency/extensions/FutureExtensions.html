    Specifies a timeout for completing this future.

    If the specified expiration time elapses, then the future is completed with the specified result.

    exp

    - the expiration time

    cb

    - the completion value on timeout

    ec

    - the implicit execution context used for executing cb

    returns

    - a new Future
